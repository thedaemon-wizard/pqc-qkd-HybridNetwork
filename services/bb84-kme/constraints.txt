# Pip CONSTRAINTS for the editable backend installs (SimQN / SeQUeNCe /
# Strawberry Fields / TNO) in the Dockerfile. Constraints pin versions WITHOUT
# installing anything and WITHOUT extras (pip rejects extras in a -c file, so
# uvicorn appears here without [standard]).
#
# Purpose: stop a transitive dependency of the heavy backends from upgrading the
# web stack past what fastapi 0.115.6 supports. fastapi 0.115.6 still passes the
# `on_startup` argument to Starlette's Router, which newer Starlette removed —
# an unpinned upgrade makes the KME crash on import with
# `Router.__init__() got an unexpected keyword argument 'on_startup'`.
fastapi==0.115.6
starlette==0.41.3
uvicorn==0.32.1
pydantic==2.10.4
# Strawberry Fields (cvqkd backend) imports the legacy `pkg_resources`, which
# setuptools >=81 removed. SeQUeNCe's install otherwise upgrades setuptools to
# 82 and breaks the cvqkd backend ("No module named 'pkg_resources'"). Hold it.
setuptools<81
